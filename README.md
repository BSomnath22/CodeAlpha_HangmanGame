# CodeAlpha_HangmanGame
#  Hangman Game in Python

A simple command-line **Hangman Game** built using Python. The player has to guess the hidden word letter by letter before the hangman is completed.

##  Features

* Random word selection
* Interactive letter guessing
* ASCII art hangman stages
* Input validation for better gameplay
* Win/Lose game conditions

## Technologies Used

* Python 3

## 📂 Project Structure

```text
hangman.py
README.md
```

## 🎯 How to Play

* The game randomly selects a hidden word.
* Guess one letter at a time.
* Correct guesses reveal letters in the word.
* Wrong guesses add parts to the hangman.
* You lose after **6 incorrect guesses**.
